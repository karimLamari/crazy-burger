# Crazy Burger

A React-based restaurant ordering application with admin panel functionality.

## Overview

Crazy Burger is a full-featured restaurant ordering system built with React and Vite. It includes a customer-facing order interface and an admin panel for menu management.

## Features

### Customer Features
- Browse menu items
- Add items to basket
- Place orders
- Toast notifications for actions

### Admin Features
- Tab-based admin panel
- Add/Edit/Delete menu items
- Manage product categories
- Real-time updates

## Technologies

- React 18
- Vite
- React Context API (state management)
- React Router
- React Toastify
- CSS Modules

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/karimLamari/crazy-burger.git
cd crazy-burger
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173)
